## 🔧 Setup Instructions

1. **Clone** the repository.
```
git clone < copied link from code >
```
2. **Change the Details** up to you.
3. **Install Node.js** initialize everything.
```
npm init
npm install express 
```
4. **Host locally**
```
node index.js
```
